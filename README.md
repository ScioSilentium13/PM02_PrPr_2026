# Учёт обращений граждан в администрации города

Программный комплекс, разработанный в рамках производственной практики по модулю ПМ02. «Осуществление интеграции программных модулей».

## Состав репозитория

- **Файлы\БД\\** – скрипты создания и наполнения базы данных Microsoft SQL Server
- **Файлы\API\CityAppealsApi\CityAppealsApi\\** – REST API (ASP.NET Core 10, Entity Framework Core)
- **Файлы\Приложение\CityAppealsApp\CityAppealsApp\\** – клиентское WPF-приложение (.NET 8, MVVM)
- **Файлы\API\CityAppealsApi\IntegrationTests\\** – интеграционные тесты API (NUnit, WebApplicationFactory)
- **Файлы\Приложение\CityAppealsApp\Tests\\** – модульные тесты клиента (NUnit, Moq)
- **Файлы\Документы\\** – руководство оператора, пояснительная записка
- **Файлы\Тестирование\\** – тест-кейсы и коллекция Postman, 
- **Документация\\** – отчёт, дневник и аттестационный лист (с копиями pdf)

## Функциональность

- Регистрация и вход граждан, операторов, исполнителей, администраторов
- Подача обращений с вложениями
- Назначение исполнителя, изменение статуса, комментирование
- Ролевой доступ: каждый пользователь видит и редактирует только разрешённые данные
- Поиск по заголовку, автору, категории
- Администрирование учётных записей

## Технологии

- **Сервер:** ASP.NET Core Web API, Entity Framework Core, Microsoft SQL Server
- **Клиент:** WPF, .NET 10, MVVM, HttpClient
- **Тестирование:** NUnit, Moq, WebApplicationFactory, Postman, ручные тест-кейсы
